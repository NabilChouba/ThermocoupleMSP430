purpose of the project is to do smart pottery oven controller
- MSP430 micro-controller
- thermocouple to sense temperature ~600-1000deg
- LCD16X2 char for display and control
- PGA112 to amplify the thermocouple voltage to make it readable by the MSP430 internal ADC

documentation and pdf of used component are in pdf directory
some picture of result and project are in picture directory
